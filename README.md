@ FACEIT CS2 NA Server Ping Benchmark

This script pings all possible FACEIT NA CS2 servers and shows the lowest ping regions for optimal matchmaking.

## Usage
1. Download `faceit_na_ping_monitor.bat`.
2. Double-click to run.
3. It will then ping every possible NA server for FACEIT CS2 (some servers may not be in the current server selection pool)
4. It will then show the latency of all NA servers (in ms)
5. It will generate a list of the top 5 lowest latency servers for optimal matchmaking
6. Finally, it will wait 45 seconds before pinging again to give a chance for the user to read the results
